Python File Handling & Data Processing Projects
This repository contains my solutions for a series of Python programming tasks focused on file I/O operations, data manipulation, and basic error handling. These exercises helped me practice working with external text files, processing user input, and generating summary reports from data.

Project Contents
Task 1: Sales Data Logger – Basic file creation and writing operations. Includes logic for both newline and comma-separated storage.

Task 2: File Reading Techniques – Demonstrates how to use .read(), .readline(), and .readlines() to pull data from files efficiently.

Task 3: Append & Maintenance – Shows how to append new data to existing files without overwriting, along with basic line counting.

Task 4: Sales Summary Report – A data processing script that reads raw text, converts it to integers, and calculates aggregate metrics like total, min, max, and average.

Task 5: Product Info System – A simple interactive script to capture user input and format it into a text-based database.

Task 6: Safe File Reading – Implements basic error checking using os.path.exists() to prevent program crashes when a file is missing.

Task 7: Discount Export Project – An end-to-end mini-project that takes a dictionary of products, applies a user-defined discount, generates a formatted text report, and appends a calculated summary at the end.

How to use this code
Each task is self-contained. You can run them in any standard Python environment. The scripts will automatically generate the required .txt files in the same directory where you run the code.

Key Concepts Practiced
File I/O Modes: Mastering 'r' (read), 'w' (write), and 'a' (append) modes.

Resource Management: Utilizing with open(...) to ensure files are closed correctly and safely.

Data Transformation: Converting between raw string input and usable Python data types (integers/floats) using list comprehensions and .strip().

Logic & Aggregation: Performing calculations on lists and dictionaries to generate summaries.
