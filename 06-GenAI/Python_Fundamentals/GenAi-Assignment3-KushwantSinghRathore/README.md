Python Practice Tasks Collection

This repository contains a set of Python functions and logic puzzles I’ve been working on to get better at handling lists, recursion, and functional programming concepts like map and filter.

Tasks Overview
Task 1: Discount Utility – A function to calculate discounted prices with a default 5% setting and a 60% cap.

Task 2: Factorial Recursion – A recursive function to calculate factorials, including error handling for negative numbers.

Task 3: GST & Combined Math – Uses lambda functions to handle tax and combined discount calculations.

Task 4: Mapping Prices – Uses map() to quickly apply tax to an entire list of prices.

Task 5: Filtering Products – Uses filter() to sort through prices based on a threshold of 500.

Task 6: Batch Processor – A combined function that discounts a list and filters the results in one go.

Task 7: Price Tracker Menu – A menu-driven program that allows users to add prices, calculate averages, and find the highest price dynamically.

How to run the code
I’ve written these as standalone scripts. You can simply copy and paste any of the blocks into your preferred Python environment (like VS Code, PyCharm, or even a Jupyter notebook).

A Note on the Logic
I spent some time making sure these functions don't crash when they encounter empty lists—I added some basic checks to return 0 instead of breaking the program. For the GST and discount calculations, I’ve ensured the math follows standard retail practices (applying discounts before tax).
